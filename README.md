# NetworkProject
네트워크 프로젝트에 사용할 깃 리포지토리


# 코드 컨벤션

## 깃 커밋 규칙
| 타입 | 내용 |
| --------- | --------------------- |
|[Feat] | 새로운 기능을 커밋할 때 사용 |
|[Fix]  | 기존의 버그를 수정할 때 사용 |
|[Refactor] | 코드 리팩토링을 할 때 사용 |
|[Docs] | 문서 부분을 수정할 때 사용 |
|[Build] | 빌드 파일 관련 부분 수정에 사용 |
|[test] | 테스트 코드(Debug)와 관련된 부분 수정에 사용 |
